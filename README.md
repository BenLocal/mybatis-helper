### Used

1. Add the JitPack repository to your build file

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

2. Add the dependency

- mybatis-helper

```xml
<dependency>
    <groupId>com.github.benlocal</groupId>
    <artifactId>mybatis-helper</artifactId>
    <version>main-SNAPSHOT</version>
</dependency>
```
